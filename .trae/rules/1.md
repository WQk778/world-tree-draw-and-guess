# AI 你画我猜智能互动游戏网站 

## 1. 项目概述
本项目是一个基于 Web 的多人在线“你画我猜”互动游戏平台，创新性地引入了 **AI（OpenAI Vision）** 作为裁判和提示生成者。用户可以在房间内进行绘画创作，AI 负责识别画作内容并生成生活化的提示词，其他用户根据提示进行竞猜。系统还包含完整的积分排行、房间管理及后台管理功能。
### 2. 技术栈
- **前端**: Vue.js (Vue 3) + Vite
- **后端**: Express.js (Node.js)
- **画布**: HTML5 Canvas
- **数据库 & 基础设施**: Supabase
  - **Auth**: 用户认证
  - **Database**: PostgreSQL (存储业务数据)
  - **Storage**: 存储用户画作图片
  - **Realtime**: 实现房间状态和游戏进程的实时同步
- **AI 服务**: OpenAI Vision API (GPT-4-Vision / GPT-4o)
